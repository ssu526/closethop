## Tagging Rules

1. Identify only the primary clothing item.
2. Ignore the background, person, mannequin, accessories, and other garments unless they are the primary item.
3. If multiple items are visible, describe only the most prominent one.
4. Describe only visually observable characteristics.
5. Never infer brand, price, gender, sustainability, comfort, quality, season, or occasion.
6. If an attribute is uncertain, do not include it as a tag.
7. Return only valid JSON matching the supplied schema.

## Tag Rules

Generate between 5 and 20 useful search tags when enough visual information is available. Fewer are acceptable when the image does not provide enough reliable detail.

Every tag must:

- contain 1–3 words

- represent only one concept

- dominant color

- visually recognizable material

- neckline or collar
